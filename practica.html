Nombre del elemento Descripcion Uso
div El elemento HTML es el contenedor genérico para el contenido de flujo. No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica estilo de alguna manera usando CSS (por ejemplo, se le aplica estilo directamente, o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal).
 El div elemento debe usarse solo cuando ningún otro elemento semántico (como article o nav) sea apropiado.
a El elemento HTML a (o elemento ancla), con su atributo href, crea un hipervínculo a páginas web, archivos, direcciones de correo electrónico, ubicaciones en la misma página o cualquier otra cosa que una URL pueda abordar El contenido dentro de cada debe indicar el destino del enlace.
head El elemento HTML head contiene información legible por máquina (metadatos) metadatos sobre el documento, como su título, scripts y hojas de estilo
body El elemento HTML body representa el contenido de un documento HTML Solo puede haber un elemento body en un documento
p The p HTML element represents a paragraph, pero los párrafos HTML pueden ser cualquier agrupación estructural de contenido relacionado, como imágenes o campos de formulario Los párrafos generalmente se representan en los medios visuales como bloques de texto separados de los bloques adyacentes por líneas en blanco y/o sangría en la primera línea
table El elemento HTML table representa datos tabulares información presentada en una tabla bidimensional compuesta de filas y columnas de celdas que contienen datos
tbody indica que comprenden el cuerpo de la tabla El elemento HTML tbody encapsula un conjunto de filas de tabla (elementos tr)
tr El elemento HTML tr define una fila de celdas en una tabla las celdas de la fila se pueden establecer mediante una combinación de elementos td
td El elemento HTML td define una celda de una tabla que contiene datos Participa en el modelo de mesa
thead El elemento HTML thead define un conjunto de filas definen el encabezado de las columnas de la tabla
style El elemento HTML style contiene información de estilo para un documento o parte de un documento. Contiene CSS se aplica al contenido del documento que contiene el elemento style
